# Recursos Visuales (Assets) - SubliAcrilico SaaS

Este directorio contiene las imágenes, logotipos e ilustraciones que dan vida a la experiencia visual premium de **SubliAcrilico**.

A continuación se detallan los archivos requeridos, sus descripciones y los tamaños recomendados para mantener una web veloz y visualmente deslumbrante (SaaS de $10,000 USD).

---

## 1. Logotipos

### `logo-horizontal.png` / `logo-horizontal.svg`
*   **Dimensiones Recomendadas**: 300 x 80 px (Fondo transparente).
*   **Descripción**: El logo principal para la barra de navegación. Debe tener tipografía estilizada y elegante basada en la fuente `Athena` con el nombre "SubliAcrilico".
*   **Colores recomendados**: `#C2AD90` (Beige arena) o `#97754D` (Bronce medio) para resaltar sobre el fondo oscuro `#0C100E`.

### `logo-monograma.png` / `logo-monograma.svg`
*   **Dimensiones Recomendadas**: 150 x 150 px (Fondo transparente).
*   **Descripción**: Un monograma simplificado con las iniciales "SA". Se utilizará para el Footer, pantallas de carga (loaders) y avatares del sistema.

### `favicon.png`
*   **Dimensiones Recomendadas**: 32 x 32 px.
*   **Descripción**: Ícono simplificado de la marca para la pestaña del navegador.

---

## 2. Fondos e Ilustraciones Principales

### `hero-bg.jpg`
*   **Dimensiones Recomendadas**: 1920 x 1080 px (Optimizado para web, menor a 300KB).
*   **Descripción**: Imagen de fondo principal de la Landing Page. Debe ser abstracta, con texturas de acrílico flotante premium, reflejos de luz y sutiles tonos dorados/bronce sobre negro oliva profundo (`#0C100E`). Aporta profundidad visual de lujo.

### `login-bg.jpg`
*   **Dimensiones Recomendadas**: 1080 x 1080 px (Cuadrado).
*   **Descripción**: Imagen lateral o de fondo difuminada para los modales y pantallas de autenticación (Login/Registro de Clientes, Administradores MAFER y DEV). Un gradiente oscuro y elegante de `#0C100E` a `#364442` con patrones sutiles.

## 3. Placeholders para el Catálogo (Muestras de Productos)

*Nota: Estos son los diseños predeterminados que usaremos para rellenar la base de datos y la vista inicial del catálogo.*

### `preview-corel.jpg`
*   **Dimensiones Recomendadas**: 600 x 600 px (Relación de aspecto 1:1).
